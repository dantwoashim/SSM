# Assurance Report Structure

## Section order

1. Executive summary
2. Tested scope
3. Scenario result matrix
4. Open findings by severity
5. Remediation guidance
6. Residual risk
7. Scope boundaries
8. Retest summary

## Audience split

- Executive summary: procurement, CTO, enterprise champion
- Scenario matrix: IAM and implementation leads
- Findings and remediation: engineering, solutions, support
- Residual risk and boundaries: security reviewers and legal
