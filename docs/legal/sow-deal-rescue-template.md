# Deal Rescue SOW Template

## Scope

- One named customer or prospect rollout
- One primary environment
- One primary IdP
- One assurance report
- One retest pass after remediation
- One readout session

## Included scenarios

- SP-initiated SSO
- IdP-initiated SSO
- JIT provisioning
- SCIM lifecycle operations in scoped flows
- Group mapping and role sync where supported
- Named customer-specific edge cases captured during intake

## Customer dependencies

- Working staging or review environment
- Access to test accounts or configuration owner
- Required metadata, credentials, or provisioning tokens
- Named deadline and customer-facing requirement set

## Deliverables

- Executive summary
- Scenario outcome matrix
- Severity-ranked findings
- Buyer-safe remediation notes
- Retest result
